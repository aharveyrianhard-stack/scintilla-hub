SPECS & DATA SOURCES
Universe: 11 SPDR sector ETFs (XLK XLC XLY XLF XLI XLB XLE XLV XLP XLU XLRE) vs benchmark SPY.
Data: Yahoo Finance spark API via same-origin relay /api/yahoo (serverless proxy on this domain — no CORS, no third parties). Fallbacks: hub relay absolute URL, direct Yahoo, public proxies.
Series: daily closes 6mo (range=6mo&interval=1d), intraday 5-min 5d, intraday 1-min 1d. Quotes ~real-time-to-15min delayed per Yahoo. Relay edge-cache 60s; page auto-refreshes every 5 min.
Timeframe buttons: 1D/5D use intraday series (1m/5m bars); 1M/3M/6M use daily. Leaders list, heatmap colors, and the % lines all follow the selected timeframe. RRG always computes on daily.
FORMULAS & COMPUTATIONS
Relative Performance %: for each sector, (close_i / close_base − 1) × 100 where base = first visible bar. Rebases live as you zoom/pan (TradingView-style). Zoom = drag the time axis; double-click axis resets.
RRG (approximation of JdK RS-Ratio/RS-Momentum): RS = 100 × sector/SPY, smoothed with EMA(10). RS-Ratio = 100 + z-score of smoothed RS over a 63-day window, then EMA(5). RS-Momentum = 100 + normalized 10-day change of RS-Ratio. Quadrants: ≥100/≥100 Leading · ≥100/<100 Weakening · <100/<100 Lagging · <100/≥100 Improving. Tail = 8 weekly samples (every 5th trading day).
Heatmap: squarified treemap. Tile size ∝ approx fund AUM ($B constants in AUM map — static, edit in source). Deselected sectors are removed and remaining tiles renormalize to fill the space. Color intensity scales with timeframe (±2.5% full color at 1D up to ±15% at 6M).
Leaders: % change over selected timeframe from daily closes; 1D = last close vs prior close (includes today's live bar during RTH).
